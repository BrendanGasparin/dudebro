# Dude Bro

Dude Bro is a one level prototype of a 2D platformer game for the Pico-8 fantasy console.  

The project is being developed in public through GitHub, Itch.io, blog posts, and YouTube videos.  

## Current Status

Version: 0.1.0  

Development Status: In development

## Features

- Player movement
- Jumping
- Collision detection
- Basic enemies
- Collectible coins and score
- A playable level

## Controls

Arrow keys - move left/right  
X - jump

## Running the Game

### Pico-8

Load the cartridge: 

```
load dudebro.p8
run
```

## Itch.io

The [latest playable version](https://brendangasparin.itch.io/dudebro) is available on Itch.io.
